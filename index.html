EFFICIENTLY DISENTANGLING CLIP FOR MULTI-OBJECT PERCEPTION
Samyak Rawlekar, Heng Ji, Minh Do
Submitted
[Abstract]
Vision-language models like CLIP excel at recognizing a single prominent object in a scene but struggle in complex multi-object settings. We identify excessive mutual feature information (MFI) as a key reason: features of one class contain substantial information about unrelated classes, causing unwanted activations. We propose DCLIP, an efficient framework that reduces MFI while adding minimal learnable parameters to a frozen VLM. DCLIP introduces an MFI Loss that regulates class feature similarity and an Asymmetric Loss (ASL) that aligns image features with disentangled text features. DCLIP reduces inter-class similarity by 30%, outperforms SOTA multi-label recognition methods on VOC2007 and COCO-14 with 75% fewer parameters, and improves zero-shot segmentation across six benchmarks.
